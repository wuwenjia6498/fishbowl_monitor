# 任务：新增“项目说明与使用指南”功能 (v5.1 User Guide)

用户希望在界面上增加一个“使用指南”入口，帮助新用户快速理解“鱼盆模型”的核心逻辑、数据来源及指标含义。

## 1. UI 设计要求

* **入口位置:** 在 `MarketHeader` 组件的右上角（或合适的操作栏位置）增加一个 **"使用指南"** 按钮。
* **样式:** 使用 `Button` 组件，`variant="ghost"`，带 `HelpCircle` 图标。
* **交互:** 点击按钮弹出 **Shadcn UI Dialog (对话框)**。
* **响应式:** 移动端只显示图标，PC 端显示 "图标 + 文字"。

## 2. 新增组件 (`components/project-intro.tsx`)

请创建一个新的客户端组件，内容需包含以下四个板块（请直接使用以下文案）：

### 2.1 核心模型：鱼盆策略 (The Strategy)
* **图标:** `TrendingUp`
* **文案:**
    > 本系统基于经典的 **20日均线 (MA20)** 趋势跟踪策略。为了过滤震荡市的假突破信号，我们引入了 **±1% 缓冲带机制**：
    > * **YES (多头):** 价格有效站上 MA20 上方 1%，确立上涨趋势。
    > * **NO (空头):** 价格有效跌破 MA20 下方 1%，确立下跌趋势。
    > * **HOLD (震荡):** 在 ±1% 缓冲区间内，维持原有状态不变。

### 2.2 数据双轨制 (Hybrid Data)
* **图标:** `Layers`
* **文案 (采用 Grid 布局展示):**
    * **🏛️ 宽基大势 (标尺):** 监控原生指数 (如沪深300)。仅作为**大盘风向标**，用来决定总仓位和择时，不提供直接交易建议。
    * **🏭 行业轮动 (标的):** 监控精选 ETF (如半导体ETF)。这是**实战交易对象**，数据经过前复权处理，信号可直接跟随。

### 2.3 关键指标解读 (Metrics)
* **图标:** `Activity`
* **文案:**
    * **区间涨幅 (Trend Perf.):** 从当前趋势信号发出那一刻起，持有至今的累计盈亏。用于验证趋势的真实性和盈利空间。
    * **偏离度 (Deviation):** 现价距离 MA20 的乖离程度。**>15%** 代表短期过热（风险），**<-15%** 代表严重超跌（机会）。

### 2.4 交互操作 (Interactions)
* **图标:** `MousePointerClick`
* **文案:**
    * 点击 **ETF 代码**：可直接复制到剪贴板，方便交易软件下单。
    * 点击 **i 图标**：查看该 ETF 的**深度投资逻辑**、规模及持仓分析。
